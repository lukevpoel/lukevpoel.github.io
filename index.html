Weather App - The Code

The purpose

The purpose of building this website was to learn ways to effectively leverage APIs.

The work

I used the openweathermap API to build a website that would take accept input from a user. The app would use the input to retrieve and display data based on the openweathermap API.

What I learned

I learned how to accept input from users, and use an API to retrieve data. I learned how to consume that data, and use it to display useful information to my user. Additionally I learned to leverage a user's local storage to enable them to easily return to previous searches.
